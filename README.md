# MBE-simulator

Key Features
Havlena–Odeh Material Balance implementation for OOIP/OGIP estimation
Industry-standard PVT correlations for oil, gas, and water property calculations
Drive mechanism analysis (solution gas, gas cap, water drive, rock & fluid expansion)
Campbell, Cole, and Havlena–Odeh diagnostic plots for reservoir evaluation
Pot Aquifer water influx model for aquifer-supported reservoirs
Reservoir performance forecasting using historical production data
Interactive Streamlit dashboard with dynamic Plotly visualizations
Historical production data upload and automated engineering analysis
